Elementos de títulos, parágrafos e formatação

A tag HEAD: Define os metadados do documetns, informações sobre o documento. É feito para o navegador reconhecer melhor o documento.
A tag BODY: É uma tag que contém o conteúdo visual, tudo que o usuário tem acesso.

Títulos

Temos algumas tags nomeadas como títulos que elas são representadas pela seguinte maneira em ordem:
H1;H2;H3;H4;H5;H6

Parágrafos

Temos também formas de formar parágrafos a tag utilizada e:
p

Formatação

No HTML podemos escrever utilizando negrito, itálico ou ambos para destacar partes de um texto.
Sendo elas:
b - para ficar em negrito
i - para fica em itálico
Não sendo muito utilizadas pois temos tags que se destacam melhor no lugar.
Elas são:
Strong - Que significa forte e usado para ficar em negrito.
Em - Que vem de 'Emphasis' significa 'Enfâse' e usado para ficar em itálico.
